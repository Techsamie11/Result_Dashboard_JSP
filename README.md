# 🎓 Result Dashboard (JSP)

A modern and responsive **Student Result Management Dashboard** developed using **Java (JSP & Servlets), JDBC, MySQL, HTML, CSS, Bootstrap, and JavaScript**. The application enables administrators to manage student records, calculate results automatically, and monitor key academic statistics through an interactive dashboard.

The system provides a clean and user-friendly interface for performing CRUD operations on student records while automatically calculating **Total Marks, Percentage, Grade, and Pass/Fail Status** based on the marks entered.

---

## 📌 Features

- 🔐 Admin Dashboard
- 👨‍🎓 Student Record Management
- ➕ Add New Student Records
- ✏️ Update Existing Student Details
- ❌ Delete Student Records
- 📋 Display Student Results
- 📊 Automatic Total Marks Calculation
- 📈 Automatic Percentage Calculation
- 🏅 Automatic Grade Generation
- ✅ Pass / Fail Result Calculation
- 📌 Dashboard Summary Cards
- 🎨 Responsive User Interface
- 🗄️ MySQL Database Integration
- ⚡ JDBC Connectivity

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/Techsamie11/Result_Dashboard_JSP.git
```

2. Import the project into Eclipse IDE.

3. Configure Apache Tomcat.

4. Create a MySQL database.

```sql
CREATE DATABASE DemoStud;
```

5. Import the SQL file.

6. Update your JDBC connection details.

7. Run the project.
